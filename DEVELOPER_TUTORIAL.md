# Gamify Photography Developer Documentation

All developer documentation has been reorganized, expanded, and split into multiple beginner-friendly files inside the `developer_docs/` folder:

1. 🚀 **[Getting Started & Setup](developer_docs/getting_started.md)**: Install prerequisites, configure Firebase, seed the database, and run both apps locally.
2. 🏛️ **[Architecture, Features & Pages Guide](developer_docs/architecture_and_logic.md)**: Deep dive into MVVM (Flutter), Feature-based React directories, and full page catalogs.
3. 🔍 **[Debugging & Troubleshooting Guide](developer_docs/debugging_guide.md)**: Breakpoints, DevTools, App Check token bypasses, and form validation error checks.
4. 🛠️ **[How to Change/Create Features & Pages](developer_docs/changing_features_and_pages.md)**: Step-by-step UI updates, routing configs, and prompting templates for the AI Coding Agent (Antigravity).
5. 🌐 **[How to Deploy Docs on GitHub](developer_docs/deploying_on_github.md)**: Steps to publish your docs via GitHub Pages (e.g. from the `/docs` folder) or static generators.
